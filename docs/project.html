Project Name: Python Quiz App with AI Tutor

The Python Quiz App is an interactive learning tool that tests Python knowledge through multiple-choice questions. When users answer incorrectly, an AI model (running locally via Ollama) generates a simple explanation of why the correct answer is right. Questions come from a database, but users can also generate new questions on any Python topic using AI.

Features

  • Multiple-choice Python questions from database
  • AI-generated explanations for wrong answers
  • Generate new questions on any Python topic
  • Tracks correct/incorrect score

Technologies

  • Backend: FastAPI (Python)
  • AI Engine: Ollama + DeepSeek-Coder (local LLM)
  • Database: SQLite
  • Frontend: HTML/CSS/JavaScript

Project Name: QIntel Application

The QIntel Application is a specialized system built with R and Shiny for event logic and response tracking. It utilizes a country detection algorithm and a SQLite backend to manage participant questions in real-time. It allows speakers to filter, classify, and display audience questions during live events.

Live Application

Read the full report: Final Project.pdf View the presentation: Presentation.pdf View the live demo:QIntel app ## AI Extension (In Progress)

I am developing an AI module that:

  • Detects duplicate questions using text embeddings
  • Classifies questions by topic
  • Suggests answers based on historical responses
  • Prevents repetitive submissions
  • Displays questions and answers live on screen

Why This Matters

Large meetings often suffer from repeated questions and missed answers. This system reduces redundancy, improves clarity, and enhances the experience for both speakers and participants.

Technologies

  • Frontend: R + Shiny
  • Database: SQLite / PostgreSQL
  • AI Engine: Embedding models for semantic similarity
  • Documentation: Quarto
